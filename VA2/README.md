# README.md

# Main file:

		"capture.py"
    
# Run:

## Command: 

		"python capture.py -q '<path-to-video-file>'"


## usage:
	capture.py [-h] [-q QUERY]

optional arguments:
  -h, --help            show this help message and exit
  -q QUERY, --query QUERY
                        Path to the query video ('capture' for VideoCapture
                        from webcam)
